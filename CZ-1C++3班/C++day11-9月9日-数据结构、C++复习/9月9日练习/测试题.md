##### 2、下面程序的时间复杂度为

```c
int BinarySearch(int* a, int n, int x)
{
	assert(a);

	int begin = 0;
	int end = n-1;
	while (begin < end)
	{
		int mid = begin + ((end-begin)>>1);
		if (a[mid] < x)
			begin = mid+1;
		else if (a[mid] > x)
			end = mid;
		else
			return mid;
	}

	return -1;
}

```

A、$O(n)$

B、$O(log_2n)$

C、$O(n^2)$

D、$O(nlog_2n)$
